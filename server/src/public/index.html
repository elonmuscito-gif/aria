Integration

From zero to verified in 3 steps

1

Register

Create your account and register your agent with a name and declared scope.

const agent = await aria.registerAgent({ name: 'invoice-processor', scope: ['read:invoices', 'write:invoices'] });
2

Track

Wrap every action with aria.track(). ARIA verifies, signs, and records — automatically.

await aria.track(agent.did, agent.secret, 'read:invoices', async () => { return await fetchInvoices(); } );
3

Prove

Show regulators, auditors, and clients cryptographic proof of everything your agent did.

{ trustScore: 94, level: "TRUSTED" }
