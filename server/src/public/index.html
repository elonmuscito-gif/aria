How It Works
THREE STEPS TO VERIFIED
01
Register Your Agent

Every agent gets a cryptographic DID. Unforgeable. Immutable. Permanent.

did:agentrust:<uuid>
npm install @ariatrust-io/aria-sdk
02
Track Every Action

Every action signed with HMAC-SHA256. Scope verified server-side. Trust score updated in real time.

await aria.track(agent.did, secret, 'read:invoices', async () => { return await fetchInvoices(); });
03
Enforce and Prove

ARIA Gate stops destructive actions. ZeroProof certifies behavior. Audit trail is cryptographically sealed.

const proof = await aria.getProof(agent.did); // { trustScore: 87, level: "TRUSTED", sealed: true }

Integration
THREE LINES TO PROTECTED
import { AriaSDK } from '@ariatrust-io/aria-sdk'; const aria = new AriaSDK({ apiKey: process.env.ARIA_API_KEY }); // Gate mode: requires human approval for destructive actions await aria.track(agent.did, agent.secret, 'delete:database', async () => { return await dropDatabase(); }, { mode: 'gate', gate: { requireApproval: ['delete:*'] } }); // ARIA blocks execution and notifies the owner // Owner receives: agent ID, action, risk level // Execution only proceeds on explicit APPROVE
import { AriaSDK } from '@ariatrust-io/aria-sdk'; const aria = new AriaSDK({ apiKey: process.env.ARIA_API_KEY }); // Light mode: track and record without blocking const result = await aria.track( agent.did, agent.secret, 'read:invoices', async () => fetchInvoices() ); // result.trustScore → 87 // result.eventId → "evt_8f2a9c" // result.sealed → true
import { AriaLangChain } from '@ariatrust-io/aria-sdk/langchain'; // Wrap your LangChain executor with ARIA const executor = AriaLangChain.wrap(agentExecutor, { agentDid: agent.did, secret: agent.secret, mode: 'gate', gate: { requireApproval: ['delete:*', 'write:*'] } }); // Every tool call is now tracked and cryptographically sealed const result = await executor.invoke({ input: 'Process invoice #4821' });
